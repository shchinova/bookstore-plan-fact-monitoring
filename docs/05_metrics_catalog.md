# Каталог метрик

> Документ содержит определения всех метрик проекта, а также формулы расчётов (поля таблиц), получателей и бизнес-смысл.

### 1.1. Выручка и объём продаж

|**Метрика**|**Определение**|**Формула / поле**|**Кому нужна**|
|---|---|---|---|
|Дневная выручка (net)|Чистые деньги от продаж за день за вычетом возвратов|SUM(sales_amount - return_amount)|Владелец — ежедневный контроль|
|MTD-выручка|Накопленная выручка с 1-го числа до сегодня|SUM за период|Владелец — темп месяца|
|Выручка предыдущего периода (prev MTD)|Тот же период прошлого месяца для сравнения|SUM за 01.prev — day.prev|Владелец — динамика|
|Валовая выручка (gross)|Продажи до вычета возвратов|SUM(sales_amount)|Аналитик — оценка возвратов|
|Выручка от возвратов|Сумма возвращённых денег|SUM(return_amount)|Аналитик — качество|
|Выручка по формату|Разбивка по eBook / Paperback / Hardcover / Audiobook / Subscription|GROUP BY format|Владелец — портфель|
|Выручка по жанру|Разбивка по Fiction, Science и т.д.|GROUP BY genre|Владелец — ассортимент|
|Выручка от промо|Продажи с применением промокода|SUM где is_promo=1|Маркетолог — ROI промо|

### 1.2. Заказы и средний чек

|**Метрика**|**Определение**|**Формула / поле**|**Кому нужна**|
|---|---|---|---|
|Число заказов (день)|Уникальные заказы за день|COUNT(DISTINCT order_id)|Владелец — трафик|
|Число заказов MTD|Накопленные заказы за месяц|COUNT MTD|Владелец — темп|
|Средний чек (день)|Выручка / заказы за день|total_revenue / total_orders|Владелец — ценообразование|
|Средний чек MTD|MTD-выручка / MTD-заказы|mtd_revenue / mtd_orders|Владелец — тренд чека|
|Средний чек prev MTD|Тот же период прошлого месяца|prev_mtd_revenue / prev_mtd_orders|Владелец — динамика|
|Размер корзины|Среднее число позиций в заказе|COUNT(sales_id) / COUNT(DISTINCT order_id)|Аналитик — upsell|

### 1.3. Маржинальность

|**Метрика**|**Определение**|**Формула / поле**|**Кому нужна**|
|---|---|---|---|
|Валовая прибыль (день)|Выручка минус себестоимость за день|net_revenue - SUM(net_qty × cost_rub)|Владелец — доходность дня|
|Валовая прибыль MTD|Накопленная прибыль за месяц|SUM MTD|Владелец — итог месяца|
|Gross margin (день)|Доля прибыли в выручке за день|gross_profit / net_revenue|Владелец — эффективность|
|Gross margin MTD|Маржа за период с начала месяца|mtd_gross_profit / mtd_revenue|Владелец — тренд маржи|
|Маржа по формату|Рентабельность каждого формата|gross_profit / net_revenue по format|Владелец — портфельное решение|
|Маржа по жанру|Рентабельность каждого жанра|GROUP BY genre|Аналитик — закупки|
|Маржа по издателю|Рейтинг издателей по прибыли|mart_margin_by_publisher|Владелец — переговоры с поставщиками|
|Целевая маржа (план)|Плановый целевой показатель|plan_margin_target|Владелец — контроль плана|

### 1.4. Планирование и выполнение плана

|**Метрика**|**Определение**|**Формула / поле**|**Кому нужна**|
|---|---|---|---|
|План на месяц (выручка)|Полный плановый объём выручки за месяц|plan_amount|Владелец — цель|
|План на дату (выручка)|Пропорциональный план: plan × (день / дней в месяце)|plan_amount_to_date|Владелец — ежедневный контроль|
|% выполнения плана на дату|Факт / план-на-дату × 100|pct_of_plan_to_date|Владелец — светофор|
|Отклонение от плана (абс.)|Факт − план-на-дату|delta_amount_to_date|Владелец — срочность|
|Прогноз закрытия месяца|Факт / прошедших дней × всего дней|fact_amount / days_passed × days_in_month|Владелец — будет ли выполнен план|
|% выполнения (исторический)|Факт полного месяца / план полного месяца|pct_of_plan в mart_plan_fact_history|Аналитик — анализ закономерностей|

### 1.5. Остатки и логистика

|**Метрика**|**Определение**|**Формула / поле**|**Кому нужна**|
|---|---|---|---|
|Текущий остаток|closing_stock на последнюю дату|closing_stock из mart_inventory_alerts|Владелец — наличие товара|
|Уровень тревоги|critical / urgent / warning по порогу 0/5/15|alert_level|Владелец — приоритет заказа|
|Дней до нуля|closing_stock / avg_daily_sales_30d|days_until_stockout|Владелец — срок заказа|
|Среднедневные продажи (30д)|Средние чистые продажи за последние 30 дней|avg_daily_sales_30d|Закупщик — норма расхода|
|Рекомендованный заказ|avg × 45 − closing_stock|recommended_order_qty|Закупщик — объём заказа|
|Потерянные продажи|Недопродажи из-за дефицита остатка|lost_sales_qty из fact_sales|Владелец — упущенная выручка|
|Пополнение склада|Объём поступивших товаров|replenishment_qty|Закупщик — контроль поставок|

### 1.6. ABC-классификация

|**Метрика**|**Определение**|**Формула / поле**|**Кому нужна**|
|---|---|---|---|
|ABC-класс|A = топ 80% выручки, B = 80–95%, C = остальные|abc_class в mart_abc|Владелец — фокус ассортимента|
|Доля в выручке (%)|Вклад продукта в общую выручку|revenue_pct|Аналитик — портфельный анализ|
|Накопленная доля (%)|Кумулятивный % для ABC-границы|cumulative_pct|Аналитик|
|Маржа продукта|(net_revenue − net_qty × cost) / net_revenue|product_margin|Владелец — рентабельность SKU|
|Активных дней|Сколько дней товар продавался|active_days|Аналитик — оборачиваемость|
|Потерянных продаж (всего)|Суммарные потери из-за дефицита за весь период|total_lost_qty|Владелец — сигнал к пополнению|

### 1.7. Подписки

|**Метрика**|**Определение**|**Формула / поле**|**Кому нужна**|
|---|---|---|---|
|Новые подписки (мес.)|Уникальные заказы подписок за месяц|new_subscriptions|Владелец — рост базы|
|Отмены (мес.)|Возвраты подписок за месяц|cancellations|Владелец — отток|
|% отмен|cancellations / total_orders × 100|cancellation_rate_pct|Владелец — удержание|
|Доля промо-подписок|Подписки по промо / все подписки|promo_pct / promo_rate_pct|Маркетолог|
|Средняя стоимость подписки|Выручка / количество подписок|avg_subscription_price|Владелец — ценообразование|
|LTV подписки|Средняя выручка на одну подписку за весь период|avg_ltv|Владелец — ценность клиента|
|Маржа подписки|(price − cost) / price|margin в mart_subscriptions_summary|Владелец|
